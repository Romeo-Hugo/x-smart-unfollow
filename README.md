# X Smart Unfollow - 智能X平台关注管理器

🚀 **快速清理X关注列表，智能识别Crypto博主，一键批量取关**

## ✨ 功能特性

- **🔍 智能分析** - AI识别Crypto领域博主
- **⚡ 批量操作** - 一键取关多个博主
- **🎯 高准确率** - 本地+云端混合分析
- **🔒 安全可控** - 用户确认后才执行操作
- **📊 进度显示** - 实时分析进度和结果
- **🌙 暗色模式** - 自动适配系统主题

## 🚀 快速开始

### 安装步骤

1. **下载插件**
   ```bash
   git clone <repository-url>
   cd x-unfollow-extension
   ```

2. **在Chrome中加载**
   - 打开 `chrome://extensions/`
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择 `x-unfollow-extension` 文件夹

3. **开始使用**
   - 打开X关注页面 (`https://x.com/<username>/following`)
   - 点击浏览器工具栏中的插件图标
   - 点击"开始分析关注列表"
   - 选择要取关的博主
   - 点击"批量取关选中博主"

## 🛠️ 技术架构

```
manifest.json          # 插件配置
├── popup.html         # 用户界面
├── popup.js           # 界面逻辑
├── content.js         # 页面分析（核心）
├── background.js      # 后台服务
├── styles.css         # 样式文件
└── icons/             # 图标资源
```

### 分析流程
1. **页面注入** - content.js注入到X页面
2. **数据提取** - 获取关注列表和博主信息
3. **本地分析** - 关键词匹配识别Crypto内容
4. **云端增强** - 调用AI API提高准确率
5. **结果展示** - 在弹出窗口中显示分析结果
6. **批量操作** - 安全执行取关操作

## ⚙️ 配置选项

### 本地设置
插件支持以下配置：
- **置信度阈值** - 默认70%（高于此值建议取关）
- **云端分析** - 开启/关闭云端AI增强
- **自动选择** - 自动选中高置信度博主
- **API限制** - 每日API调用次数限制

### 云端API集成
支持以下AI服务：
- OpenAI GPT API
- Google AI API
- 自定义分类API

## 🐛 故障排除

### 常见问题

1. **插件不显示图标**
   - 检查是否成功加载扩展
   - 刷新X页面
   - 重新加载扩展

2. **分析不到关注者**
   - 确保在X关注页面
   - 页面完全加载后再点击分析
   - 尝试滚动页面加载更多关注者

3. **取关操作失败**
   - X页面结构可能已更新
   - 检查网络连接
   - 手动刷新页面重试

### 调试模式
在content.js中取消注释console.log语句进行调试：
```javascript
// 启用详细日志

```
